먼저 sql폴더에서 테이블을 만들어주시고 서버를 실행한다음에 주소창에 
'http://localhost:9090/notice/list/1'를 치시면 됩니다.

등록버튼으로 등록하면됩니다.

제목을 누르시면 조회 페이지로 넘어갑니다. 

조회에 수정과 삭제 버튼이있습니다. 

수정 삭제 등록을 완료하면 자동으로 목록 첫번째 페이지로 넘어갑니다.

목록 페이지는 10개씩이 1페이지이며 10페이지가 되면 이전이 생깁니다.

파일업로드도 여러개 가능합니다.
